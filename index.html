Welcome to Image Processing in Java.

Image processing is referred to processing of a 2D picture by a computer. Basic definitions:

An image defined in the “real world” is considered to be a function of two real variables, for example, a(x,y) with a as the amplitude (e.g. brightness) of the image at the real coordinate position (x,y).

Here I tried to implement some basic algorithm of image processing using Java. if you like, you can use this source for your further study or any use.

to clone this repository follow the commands given below - 

$ cd your_repo_directory
$ git clone git@github.com:rokon12/image-processing.git

here I have implemented the following algorithms -

  1. Negative
  2. Grayscale
  3. Histogram Equalize
  4. Point Detect
  5. Edge Detect
  6. Corner Detect
  7. Boundary Extract & Compare
